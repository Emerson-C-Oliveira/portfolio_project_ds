Previsão de Vendas

Este projeto foi desenvolvido com o objetivo de prever as vendas futuras de uma farmácia com base em dados históricos. Utilizando técnicas avançadas de ciência de dados e machine learning, o projeto visa fornecer insights valiosos que podem ajudar na tomada de decisões estratégicas, como planejamento de estoque e campanhas de marketing.

O projeto envolve uma série de etapas, incluindo a coleta e a limpeza dos dados, a análise exploratória, a seleção de características relevantes, a construção e a avaliação de modelos preditivos. Diversos algoritmos de machine learning foram testados para identificar o modelo com melhor desempenho.

O modelo final, baseado em XGBoost, teve um desempenho notável, com uma margem de erro de 9,52% em relação aos valores reais de vendas. Este resultado indica uma alta capacidade do modelo em capturar padrões de vendas e fornecer previsões confiáveis, contribuindo significativamente para a otimização do estoque e das estratégias de marketing da farmácia.

As ferramentas utilizadas foram:

  • Linguagem de Programação: Python 3.10.9.
  • Bibliotecas de Data Science: Pandas, NumPy, Scikit-learn.
  • Visualização de Dados: Matplotlib, Seaborn.
  • Modelagem Preditiva: Random Forest Regressor, XGBoost Regressor, LSTM (Long Short-Term Memory).
  • Métrica de Performance Final: MAPE (Mean Absolute Percentage Error).
  • Validação dos Modelos: Cross-Validation, GridSearchCV.
  • Ambiente de Desenvolvimento: Jupyter Notebook, VS-Code
  • Controle de Versão e Hospedagem: Git, GitHub.

Propensão de Compra

Este projeto foi desenvolvido para prever a propensão de clientes a realizar uma compra com base em diversos dados comportamentais e demográficos. Utilizando técnicas avançadas de machine learning, o objetivo do projeto é ajudar a identificar clientes em potencial e otimizar estratégias de marketing direcionadas.

O projeto envolve uma série de etapas, incluindo a coleta e limpeza dos dados, análise exploratória, seleção de características relevantes, construção e avaliação de modelos preditivos. Diversos algoritmos de machine learning foram testados para identificar o modelo com melhor desempenho.

O modelo final, baseado no Random Forest Classifier teve um desempenho notável, com uma acurácia de 95% na previsão dos clientes mais propensos a adquirir o novo produto. Este resultado demonstra a capacidade do modelo de identificar padrões nos dados que são indicativos da probabilidade de compra, permitindo uma segmentação mais eficiente e campanhas de marketing mais eficazes.

As ferramentas utilizadas foram:

  • Linguagem de Programação: Python 3.10.9.
  • Bibliotecas de Data Science: Pandas, NumPy, Scikit-learn.
  • Visualização de Dados: Matplotlib, Seaborn.
  • Modelagem Preditiva: Random Forest Classifier, XGBoost Classifier, Logistic Regression.
  • Métrica de Performance Final: Acurácia (Accuracy).
  • Validação dos Modelos: Cross-Validation, GridSearchCV.
  • Ambiente de Desenvolvimento: Jupyter Notebook, VS-Code.
  • Controle de Versão e Hospedagem: Git, GitHub.
